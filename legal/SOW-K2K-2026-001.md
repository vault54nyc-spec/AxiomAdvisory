## 1. PROJECT OVERVIEW

Following a 2.5-hour intake consultation conducted on March 22, 2026, Axiom has scoped a full-scale digital infrastructure engagement for Client. What began as a website refresh request has been expanded into a complete platform build encompassing brand identity, public website, authenticated student portal, and advisor admin dashboard.

This SOW governs the specific scope, fees, timeline, and deliverables for this engagement as selected and agreed upon by the Parties.

## 2. SCOPE OF SERVICES

### 2.1 Non-Negotiable Services (Required — Included in All Scopes)

**A. Technical Infrastructure** — $1,500
- Custom domain registration and full DNS configuration
- Production hosting setup (Vercel)
- Supabase database, authentication system, and file storage architecture
- Google Workspace professional branded email (one user, @[client domain])
- SSL security certificates across all subdomains
- Environment configuration (development, staging, production)

**B. Project Management** — Pro-Rated Fee ($300 minimum / $1,000 maximum)
- Dedicated project manager for the full engagement duration
- Weekly progress reports and milestone tracking
- Centralized client communication hub
- Timeline management and scope change documentation
- Pro-rated fee is calculated as: (Selected Optional Scope ÷ Full Optional Scope) × $1,000, with a minimum floor of $300 regardless of scope selected.
